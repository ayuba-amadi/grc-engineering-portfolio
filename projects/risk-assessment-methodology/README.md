# Risk Assessment Methodology

## Overview
This project demonstrates a full risk assessment process, starting from technical vulnerability scanning and ending with business-focused risk analysis.

## Scope
The assessment covers:
- Service discovery
- Vulnerability identification
- Risk prioritization
- Quantitative risk analysis (SLE, ALE)
- Business Impact Analysis (BIA)
- Risk treatment strategy
- Control mapping (NIST SP 800-53)

## Tools Used
- Nmap
- Nikto
- Nuclei

## Framework Alignment
- NIST Cybersecurity Framework
- NIST SP 800-53
- CIS Controls
- PCI DSS
- OWASP Top 10

## Structure
- documentation: full report and summaries
- risk-analysis: structured analysis outputs
- technical-evidence: raw scan results
- visuals: charts and graphs
- screenshots: execution evidence
- artifacts: supporting files
