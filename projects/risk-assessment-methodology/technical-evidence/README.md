## Technical Evidence

### Overview
This section contains raw and unmodified outputs generated from security assessment tools used during the vulnerability scanning process.

The purpose of this section is to provide verifiable and auditable evidence supporting the findings, risk analysis, and conclusions presented in this assessment.

---

### Scope of Evidence

The following tool outputs are included:

- **Nmap:** Network scanning results identifying open ports, services, and system characteristics  
- **Nikto:** Web server vulnerability assessment results  
- **Nuclei:** Automated vulnerability detection with CVE references and severity classification  

---

### Integrity and Traceability

All files in this directory are preserved in their original format to ensure:

- Accuracy of technical findings  
- Traceability between raw data and reported risks  
- Reproducibility of the assessment process  

No modifications have been made to scan outputs beyond format conversion (e.g., XML to HTML for readability).

---

### Relevance to Risk Analysis

The evidence provided in this section directly supports:

- Identification of vulnerable services and configurations  
- Mapping of technical findings to risk levels  
- Alignment with security frameworks such as CIS Controls and NIST standards  
- Justification of business impact and risk treatment decisions  

---

- Nmap
