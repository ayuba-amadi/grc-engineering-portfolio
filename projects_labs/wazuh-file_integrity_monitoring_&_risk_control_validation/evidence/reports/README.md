# Compliance Reports

This folder contains compliance reports generated from Wazuh.

These reports demonstrate how security events map to regulatory frameworks.

## Available Reports

- PCI_DSS_report.pdf
- HIPAA_report.pdf
- NIST_report.pdf
- GDPR_report.pdf
- MITRE_report.pdf
- TSC_report.pdf

## Purpose

These reports provide:

- Evidence for audits
- Mapping between alerts and compliance requirements
- Insight into how technical controls support governance

## Key Insight

The same technical event (file change) is mapped across multiple frameworks.

This reduces manual compliance effort and strengthens audit readiness.
