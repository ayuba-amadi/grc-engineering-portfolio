# Evidence

This folder contains all supporting evidence generated during the lab.

The purpose of this evidence is to:

- Validate control effectiveness
- Support audit requirements
- Provide traceability for all activities

## Structure

### Screenshots
Contains visual proof from:

- Wazuh Dashboard
- Alerts view
- File activity logs
- System interactions

### Reports
Contains exported compliance reports from Wazuh:

- PCI DSS
- HIPAA
- NIST 800-53
- GDPR
- MITRE ATT&CK
- TSC

All evidence aligns with the simulated risk scenario.
