# Screenshots

## Description

This folder contains visual evidence supporting the findings in the report.

## Subfolders

### reconnaissance/

* Ping results
* Nmap scans

### scanning/

* Burp Suite scan dashboard
* Identified vulnerabilities

### exploitation/

* Proof of Broken Access Control
* Plaintext credential capture
* Authentication testing

## Purpose

Screenshots serve as:

* Visual validation of findings
* Supporting evidence for reports
* Demonstration for presentations

## Best Practice

Each screenshot should:

* Be clearly named
* Match report references
* Show timestamp or context where possible
