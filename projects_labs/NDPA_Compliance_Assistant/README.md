# NDPA Compliance Monitor (Copilot Studio Agent)

## Overview

NDPA Compliance Monitor is an AI-powered compliance assessment agent built using Microsoft Copilot Studio. It helps organizations evaluate their readiness under the Nigeria Data Protection Act (NDPA) 2023 through a structured, guided assessment.

The agent identifies compliance gaps, classifies risk levels, and generates a clear, audit-style readiness report with actionable recommendations.

This project demonstrates practical application of Governance, Risk, and Compliance (GRC) using AI.

---

## Key Features

- Structured NDPA compliance assessment (step-by-step)
- Covers 10 critical NDPA operational areas
- Real-time gap identification
- Risk scoring and classification (Low, Medium, High)
- Audit readiness evaluation
- Practical, actionable recommendations
- Clean, professional compliance report output

---

## Compliance Scope

The agent evaluates the following NDPA-aligned areas:

1. Applicability and organizational role  
2. Governance and accountability  
3. Data Protection Officer (DPO) readiness  
4. Registration and Major Importance status  
5. Compliance Audit Return (CAR) readiness  
6. Lawful basis, consent, and transparency  
7. DPIA and high-risk processing  
8. Security, training, and breach response  
9. Vendor and processor management  
10. Cross-border data transfer compliance  

---

## How It Works

1. The agent guides the user through a structured assessment.
2. Each question includes:
   - A short explanation  
   - A simple practical example  
3. Responses are captured as:
   - Yes (Compliant)  
   - No (Gap)  
   - Not sure (Partial gap)  
4. A scoring model is applied:
   - Yes = 0  
   - Not sure = 1  
   - No = 2  
5. A final report is generated including:
   - Risk level  
   - Identified gaps  
   - Regulatory and business impact  
   - Audit readiness  
   - Recommended actions

## Deployment (Import Solution)

To deploy this agent in your environment:

1. Go to: https://make.powerapps.com  
2. Select the correct **Environment** (same one used for development or your target environment)  
3. Navigate to **Solutions**  
4. Click **Import**  
5. Upload the provided solution `.zip` file  
6. Complete the import process  
7. Open the imported solution  
8. Locate the agent (NDPA Compliance Monitor)  
9. Open in **Copilot Studio**  
10. Click **Publish**  
11. Enable a channel:
    - Demo Website (for quick access)  
    - Microsoft Teams (for organizational use)  

After this, the agent is ready for use.
