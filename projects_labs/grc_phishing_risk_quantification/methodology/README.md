# Methodology

This folder describes the analytical methods and formulas used in the phishing risk quantification project.

It provides a clear explanation of how raw phishing simulation data was transformed into measurable behavioral metrics, organizational risk exposure, and financial impact.

---

## Contents

### phishing-risk-methodology.md

This document contains the full methodology used in the analysis, including:

- Behavioral metric calculations  
- Risk projection approach  
- Financial modeling (SLE, ARO, ALE)  
- Control evaluation logic  
- Statistical analysis (confidence intervals)  

---

## Purpose

The methodology ensures that the analysis is:

- Transparent  
- Reproducible  
- Consistent across all phases  

It allows readers to understand exactly how results were derived and how similar analyses can be conducted in other environments.

---

## Note

All formulas and assumptions described here are directly applied in the documentation and supported by datasets in the technical evidence folder.
