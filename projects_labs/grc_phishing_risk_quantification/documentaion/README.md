# Documentation

This folder contains the complete phishing risk analysis presented in both structured Markdown files and a compiled final report.

The purpose of this documentation is to provide a clear, traceable, and professional presentation of how phishing simulation data was analyzed, quantified, and translated into financial risk and executive-level decisions.

---

### 1. Full Report (Recommended)

- full-report.pdf  

This document contains the complete analysis in a single, continuous format. It is designed for:

- Executive review  
- Formal submission  
- Quick understanding of the entire project  

---

### 2. Phase-Based Documentation

The analysis is also organized into four structured phases for step-by-step understanding:

- phase1-basic-metric-calculations.md  
- phase2-advanced-risk-quantification.md  
- phase3-comprehensive-risk-assessment.md  
- phase4-executive-reporting.md  

Each phase builds on the previous one and maintains full transparency of calculations and reasoning.

---

## Structure of Analysis

### Phase 1 – Basic Metric Calculations  
Focuses on analyzing employee behavior during phishing simulations by calculating open rates, click rates, and credential submission rates.

### Phase 2 – Advanced Risk Quantification  
Transforms behavioral metrics into organizational risk exposure and financial impact using risk projection and financial modeling.

### Phase 3 – Comprehensive Risk Assessment  
Evaluates and compares campaign risks, introduces financial exposure analysis, and validates findings using statistical methods.

### Phase 4 – Executive Reporting  
Presents the results in an executive format, including financial justification, control evaluation, and strategic recommendations.
