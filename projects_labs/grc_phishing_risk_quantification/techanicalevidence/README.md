# Technical Evidence

This folder contains all supporting materials used in the phishing risk analysis.

It is structured to separate visual outputs from analytical datasets for clarity and traceability.

---

## Structure

### charts/
Contains all visual outputs used in the analysis:
- Phishing funnel
- Campaign comparison charts
- Financial exposure charts
- Risk reduction and ROI visuals

### data/
Contains all datasets and calculated values:
- Behavioral metrics
- Risk projections
- Financial analysis
- Risk assessment matrix

---

## Purpose

This section ensures that all results presented in the documentation are:

- Transparent  
- Reproducible  
- Traceable to source data  

All charts and conclusions in the report are derived from these files.
