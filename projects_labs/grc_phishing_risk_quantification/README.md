# GRC Phishing Risk Quantification and Financial Impact Analysis

## Overview

This project presents a comprehensive cybersecurity risk analysis focused on phishing attacks within an organizational environment. The objective is to quantify human-related security risks and translate them into measurable financial impact using structured Governance, Risk, and Compliance (GRC) methodologies.

The analysis is based on phishing simulation data from three campaigns conducted within a simulated organization (GlobalTech Inc.). The results demonstrate how user behavior can directly contribute to large-scale financial exposure.

---

## Key Findings

- Maximum credential compromise rate: 30%
- Estimated compromised accounts (organization-wide): 600 users
- Expected breach incidents: 150
- Annualized Loss Expectancy (ALE): $652,500,000

The findings indicate that phishing attacks represent a critical business risk, driven primarily by human behavior.

---

## Methodology

The analysis follows a structured four-phase approach:

1. Behavioral Analysis (Phishing Metrics)
2. Risk Quantification (Organizational Exposure)
3. Risk Assessment (Financial + Statistical Validation)
4. Executive Reporting (Decision Support)

Each phase builds on the previous one to ensure logical consistency and traceability.

## Business Impact

This project demonstrates how cybersecurity risks can be:

- Quantified using real data
- Translated into financial exposure

It highlights that effective cybersecurity decision-making must be based on measurable impact rather than assumptions.
