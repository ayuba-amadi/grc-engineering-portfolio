# Phase 2: Control Evaluation

## 1. Introduction

In this phase, I evaluated security control options based on their cost, effectiveness, and financial impact on risk reduction.

---

## 2. Control Mapping

| Control | Target Risk |
|--------|------------|
| API Security Gateway | API Authentication Bypass |
| Web Application Firewall (WAF) | API + Database Injection |
| Multi-Factor Authentication (MFA) | Session Hijacking |

---

## 3. Cost-Benefit Analysis

### Web Application Firewall (WAF)
Cost = $150,000  
Effectiveness = 75%  
Risk Reduction = $24,000,000  
ROI = 159  
Payback = 0.006 years  

---

### Multi-Factor Authentication (MFA)
Cost = $200,000  
Effectiveness = 95%  
Risk Reduction = $2,850,000  
ROI = 13.25  
Payback = 0.07 years  

---

### API Security Gateway
Cost = $350,000  
Effectiveness = 90%  
Risk Reduction = $675,000  
ROI = 0.93  
Payback = 0.52 years  

---

## 4. Summary

| Control | Cost | Risk Reduction | ROI |
|--------|------|---------------|-----|
| WAF | $150,000 | $24,000,000 | 159 |
| MFA | $200,000 | $2,850,000 | 13.25 |
| API Gateway | $350,000 | $675,000 | 0.93 |

---

## 5. Key Findings

- WAF provides the highest financial benefit  
- MFA provides strong protection with fast return  
- API Gateway provides lower financial impact
