# Data

This folder contains all datasets and calculated values used in the analysis.

## Contents

- risk_calculations.csv  
  → Risk metrics (SLE, ARO, ALE) for identified vulnerabilities  

- control_analysis.csv  
  → Control cost, effectiveness, risk reduction, ROI, and payback  

- risk_control_matrix.csv  
  → Mapping of risks, controls, residual risk, and financial impact  

## Purpose

These datasets provide:

- Transparency of calculations  
- Traceability of results  
- Support for all conclusions presented in the documentation  

All values are derived from the project’s risk quantification and control evaluation process.
