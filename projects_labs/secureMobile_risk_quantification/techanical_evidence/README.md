# Technical Evidence

This folder contains all supporting materials used in the cybersecurity risk analysis and control evaluation.

It is structured to clearly separate visual outputs from analytical datasets, ensuring clarity, traceability, and professional presentation.

---

## Folder Structure

### charts/

This folder contains all visual outputs generated during the analysis.

#### Phase 1: Risk Analysis
- ALE Comparison Bar Chart  
- Risk Distribution Pie Chart  
- Risk Heat Map  

#### Phase 2: Control Evaluation
- Cost vs Effectiveness Scatter Plot  
- ROI Comparison Chart  
- Risk Reduction Line Graph  

#### Phase 3: Executive Reporting
- Risk Exposure Before vs After (Double Bar Chart)  
- Control Investment Breakdown (Stacked Bar Chart)  
- ROI Comparison (Horizontal Bar Chart)  

---

### data/

This folder contains all datasets and calculated values used in the analysis.

- risk_calculations.csv
  → Contains SLE, ARO, and ALE values for all vulnerabilities  

- control_analysis.csv  
  → Contains cost, effectiveness, risk reduction, ROI, and payback calculations  

- risk_control_matrix.csv  
  → Executive-level mapping of risks, controls, residual risk, and financial impact  

---

## Purpose

This section ensures transparency and reproducibility of the analysis by providing both:

- Raw data (CSV files)  
- Supporting visual evidence (charts)  

All results presented in the documentation are directly derived from these files.

---

## Analytical Value

The materials in this folder support:

- Risk prioritization  
- Financial risk quantification  
- Control effectiveness evaluation  
- Executive decision-making  

---

## Note

All charts are designed to be:

- Professionally formatted  
- Clearly labeled (titles, axes, legends)  
- Suitable for executive-level reporting  

All datasets are structured to allow independent verification of calculations and conclusions.
