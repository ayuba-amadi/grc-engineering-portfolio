# GRC Engineering Portfolio

## Overview
This repository presents a structured portfolio of cybersecurity governance, risk assessment, and compliance engineering work.

The projects demonstrate the ability to translate technical findings into business-focused risk decisions using recognized industry frameworks.

## Core Capabilities
- Vulnerability Assessment and Analysis
- Risk Identification and Prioritization
- Quantitative Risk Analysis (SLE, ALE)
- Business Impact Analysis (BIA)
- Risk Treatment Strategy Development
- Security Control Mapping (NIST SP 800-53)
- Governance and Compliance Design
- And much more

## Frameworks Referenced
- NIST Cybersecurity Framework (CSF)
- NIST SP 800-53
- ISO/IEC 27001
- CIS Controls v8
- PCI DSS
- OWASP Top 10

## Portfolio Structure
The repository is organized into structured project areas:

- Risk Assessment Methodology
- Access Control Governance
- Security Governance Architecture
- Security Awareness Program

Each project contains:
- Documentation
- Risk Analysis
- Technical Evidence
- Visualizations
- Supporting Artifacts
